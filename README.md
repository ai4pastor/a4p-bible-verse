# A4P 성경구절 (a4p-bible-verse)

옵시디언 볼트의 성경 구절 노트를 장절 참조로 검색해서, 작성 중인 노트에 **인용 콜아웃 + wikilink** 형태로 삽입하는 플러그인입니다.

구절당 1개 노트(`요3_16.md`)가 `구약/신약 → 책 폴더` 구조로 들어 있는 성경 데이터셋을 사용합니다. 각 구절 노트에는 5개 역본(새번역·개역개정·쉬운성경·NIV·KJV)이 `> [!quote] 역본명` 콜아웃으로 들어 있어야 합니다.

## 설치 (BRAT)

1. 커뮤니티 플러그인에서 **BRAT**(Obsidian42 - BRAT)를 설치·활성화합니다.
2. BRAT 설정 → `Add Beta plugin` → 이 저장소 주소를 입력합니다.
3. 커뮤니티 플러그인 목록에서 **A4P 성경구절**을 활성화합니다.

## 초기 설정

1. 설정 → **A4P 성경구절** → **성경 폴더 경로**에 볼트 내 성경 폴더 경로를 입력합니다.
   - 예: `100. notes/170. 성경` (이 폴더 아래에 `구약`, `신약` 폴더가 있어야 합니다)
2. **검증** 버튼을 눌러 66권 인식과 샘플 구절 읽기가 통과하는지 확인합니다.
3. **기본 역본**을 선택합니다 (기본값: 새번역).

## 사용법

### 검색·삽입 모달

커맨드 팔레트에서 **"성경 구절 검색·삽입"**을 실행합니다 (단축키 지정 권장, 예: `Cmd+Shift+B`).

```
검색 입력:  요3:16 / 요3:16-20 / 요한복음 3장 16절 / 시23편
```

| 키 | 동작 |
|---|---|
| `Enter` | 체크된 절 전체 삽입 후 닫기 |
| `↓` / `↑` | 목록 이동 (`↓`로 목록 진입) |
| `Space` (목록에서) | 절 선택/해제 |
| `Cmd+Enter` | 현재 절 1개만 삽입하고 모달 유지 — 이어서 다른 참조 입력 가능 |
| `Tab` / `Shift+Tab` | 역본 전환 |
| `Esc` | 목록 → 입력창 → 닫기 |

### 에디터 자동완성

노트에서 `;;요3:16`처럼 **트리거 + 참조**를 입력하면 역본 5개가 제안되고, 선택하면 바로 콜아웃으로 치환됩니다. (설정에서 끄거나 트리거 변경 가능)

### 병렬 삽입 — 이중 언어 설교용

해외 사역 등으로 한글·영문 본문이 함께 필요하면 **병렬 트리거**(기본 `;;;`)를 사용합니다:

```
;;;요3:16-17
```

설정에 등록해둔 역본 쌍(기본: 새번역 + NIV)이 **절마다 교차**로 들어갑니다:

```markdown
> [!quote] [[요3_16|요한복음 3:16-17]] (새번역 · NIV)
> [[요3_16|16]] 하나님이 세상을 이처럼 사랑하셔서 …
> _For God so loved the world that he gave his one and only Son, …_
> [[요3_17|17]] 하나님께서 아들을 세상에 보내신 것은 …
> _For God did not send his Son into the world to condemn the world, …_
```

역본 쌍과 트리거는 설정 → **병렬 삽입 (이중 역본)**에서 변경할 수 있습니다.

### 삽입 형식

단일 절:

```markdown
> [!quote] [[요3_16|요한복음 3:16]] (새번역)
> 하나님이 세상을 이처럼 사랑하셔서 독생자를 주셨으니, …
```

범위 (절마다 구절 노트로 wikilink가 걸려 백링크·그래프 연결이 유지됩니다):

```markdown
> [!quote] [[요3_16|요한복음 3:16-18]] (새번역)
> [[요3_16|16]] 하나님이 세상을 이처럼 사랑하셔서 …
> [[요3_17|17]] 하나님께서 아들을 세상에 보내신 것은 …
> [[요3_18|18]] 아들을 믿는 사람은 심판을 받지 않는다 …
```

## 지원하는 참조 문법

| 입력 | 결과 |
|---|---|
| `요3:16`, `요3_16`, `요3장16절` | 단일 절 |
| `요한복음 3장 16절` | 단일 절 (정식 책 이름) |
| `요3:16-20`, `요3:16~20` | 절 범위 |
| `요3`, `요3장`, `시23편` | 장 전체 |
| `요일4:9` | 요한1서 4:9 (약자 자동 구분) |

- 없는 절 범위는 자동으로 잘라서 안내합니다 (예: `요3:35-40` → 3:35-36).
- 장을 넘는 범위(`요3:36-4:2`)는 지원하지 않습니다 — 장별로 나눠 삽입해주세요.
